# Официальный веб-сайт ТОО "ОВЕРКОМ"

**ТОО "ОВЕРКОМ"** — лидирующая сервисная компания в горнодобывающей отрасли Казахстана (г. Караганда), специализирующаяся на геологоразведке, буровых и взрывных работах.

## 📌 О проекте
Данный проект представляет собой многостраничный корпоративный сайт, разработанный в рамках производственной практики по специальности "Software Engineering". Сайт спроектирован как высокотехнологичное решение для демонстрации производственных мощностей предприятия, привлечения партнеров и автоматизации сбора заявок.

## 🏗 Структура сайта (5 разделов)
1. **Главная (`index.html`)** — Инфографика (статистика 12+ лет), краткий обзор направлений и интерактивная фотогалерея объектов.
2. **О компании (`about.html`)** — История становления (с 2011 года), миссия и полные юридические реквизиты (БИН, ОКЭД).
3. **Услуги (`services.html`)** — Детальное описание компетенций: RC-бурение, ГИС-моделирование (Micromine), БВР и инженерный консалтинг.
4. **Карьера (`career.html`)** — HR-модуль с активными вакансиями и формой отправки резюме (поддержка загрузки CV).
5. **Контакты (`contacts.html`)** — Интерактивная карта Яндекс (пр. Шахтеров, 25), прямые каналы связи и форма обратной связи.

## 🛠 Технологический стек
- **HTML5:** Семантическая верстка для обеспечения доступности и SEO.
- **CSS3:** - **Grid & Flexbox** для построения сложных адаптивных сеток.
    - **Custom Properties (Variables)** для управления глобальной цветовой схемой.
    - **Keyframes & Transitions** для микровзаимодействий и анимаций.
- **JavaScript (ES6+):** - Логика мобильной навигации (Hamburger menu).
    - **Intersection Observer API** для плавной анимации элементов при скролле.
    - Валидация форм и обработка пользовательского ввода.
- **Интеграции:** FontAwesome 6.4.0 (иконки), Yandex Maps API (интерактивный виджет).

## 🎨 Дизайн и палитра (Industrial Dark)
- **Primary Black (`#0f1113`):** Символизирует уголь, сталь и надежность.
- **Accent Red (`#e63946`):** Сигнальный цвет промышленной безопасности и энергии.
- **Background Grey (`#f8f9fa`):** Обеспечивает высокую читаемость технической документации.

## 🚀 Основные возможности
- **Адаптивность:** Корректное отображение на смартфонах, планшетах и десктопах.
- **UX/UI:** Плавная прокрутка, подсветка активных разделов меню и hover-эффекты для кнопок.
- **Функционал:** Рабочая форма обратной связи, фильтрация изображений в галерее и реальная геолокация офиса.
