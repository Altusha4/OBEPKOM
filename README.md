# Сайт ТОО "ОВЕРКОМ"

Официальный сайт компании ТОО "ОВЕРКОМ" - услуги для добывающей промышленности.

## Текущий статус
✅ Базовая структура HTML  
✅ Навигационное меню  
✅ Баннер "Написать нам"  
✅ Поисковая строка  
✅ Секция "О компании"  
✅ Секция "Услуги" (4 основные услуги)  
✅ Секция "Фотогалерея"  
✅ Секция "Клиенты"  
✅ Адаптивный дизайн  
✅ CSS Grid для сеток

## Структура сайта
1. **Шапка** - логотип и навигация
2. **Баннер "Написать нам"** - контактная информация
3. **О компании** - основные данные компании
4. **Услуги** - 4 карточки с описанием услуг
5. **Фотогалерея** - примеры работ с подписями
6. **Клиенты** - категории клиентской базы
7. **Подвал** - копирайт и БИН

## Новые разделы в этом коммите

### Фотогалерея
- Три примера фотографий с подписями
- Адаптивная сетка с эффектами наведения
- Заголовки для каждой фотографии

### Клиенты
- Три категории клиентов:
    1. Горнодобывающие компании
    2. Промышленные предприятия
    3. Строительные организации
- Детализированные списки в каждой категории
- Чистая визуальная структура

## Технологии
- HTML5 (семантическая верстка)
- CSS3 (Grid, Flexbox, анимации)
- Адаптивный дизайн (мобильные устройства, планшеты, десктопы)
