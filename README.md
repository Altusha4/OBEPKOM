# Сайт ТОО "ОВЕРКОМ"

Официальный сайт компании ТОО "ОВЕРКОМ" - услуги для добывающей промышленности.

## О проекте
Учебный проект для практики в университете. Полностью функциональный фронтенд сайта с адаптивным дизайном и JavaScript интерактивностью.

## Структура сайта
1. **Шапка** - логотип и навигация
2. **Баннер "Написать нам"** - контактная информация
3. **О компании** - основные данные компании
4. **Услуги** - 4 карточки с описанием услуг
5. **Фотогалерея** - примеры работ с подписями
6. **Клиенты** - категории клиентской базы
7. **Техника** - оборудование компании
8. **Контакты** - контакты и форма обратной связи
9. **Подвал** - копирайт и БИН

## Технологии
- HTML5 (семантическая верстка)
- CSS3 (Grid, Flexbox, анимации, медиа-запросы)
- JavaScript (ES6+)
- Адаптивный дизайн (мобильные устройства, планшеты, десктопы)

## JavaScript Функциональность

### 1. Плавная прокрутка
- Плавный переход по якорным ссылкам
- Учет высоты фиксированной шапки
- Предотвращение стандартного поведения ссылок

### 2. Валидация формы
- Проверка обязательных полей
- Валидация email формата
- Динамическое отображение ошибок
- Очистка формы после успешной отправки

### 3. Подсветка навигации
- Автоматическое определение активного раздела
- Изменение стиля активной ссылки
- Обновление при прокрутке страницы

### 4. Поиск по сайту
- Поиск по тексту разделов
- Подсветка найденных разделов
- Уведомление о результатах поиска

## Особенности реализации
- Единый стиль карточек во всех разделах
- Плавные анимации при наведении
- Четкая иерархия заголовков
- Оптимизированная читаемость текста
- Постепенная деградация для старых браузеров
- Чистый, хорошо откомментированный код
